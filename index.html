Welcome to GitHub Pages.

FastDCS 是一个可以进行快速部署的开源分布式计算系统。系统设计目标是具备高可用性、高性能、可扩展性和一致性。

可以在可能出现以下异常情况的环境中运行:


宕机：部分Master和Worker服务器宕机。
网络异常：消息丢失，消息乱序，数据错误，不可靠的TCP。
一致性：存储的数据丢失。
